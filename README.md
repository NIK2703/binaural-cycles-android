# Binaural Cycles

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

An Android app for creating and listening to dynamic binaural beats aligned with human circadian rhythms by scheduling carrier and beat frequency changes throughout the day.

## Features

- **Customizable presets** — Create and listen to binaural beat presets with precise carrier and beat frequency distribution throughout the day
- **Interpolation modes** — Linear or cubic interpolation between frequency points
- **Auto channel swap** — Periodic left/right channel switching for balanced stimulation
- **Volume normalization** — Automatic volume balancing between channels based on frequency ratio

## How Binaural Beats Work

When two slightly different frequencies are played in each ear, your brain perceives a third "beat" at the difference between them. For example, if 400 Hz plays in the left ear and 410 Hz in the right, you perceive a 10 Hz beat.

Different beat frequencies are associated with different mental states:

| Frequency Range | Brain State |
|-----------------|-------------|
| Delta (0.5–4 Hz) | Deep sleep, healing |
| Theta (4–8 Hz) | Meditation, creativity |
| Alpha (8–14 Hz) | Relaxation, calm focus |
| Beta (14–30 Hz) | Active thinking, concentration |
| Gamma (30–100 Hz) | Higher cognitive functions |

> **Note:** Binaural beats require stereo headphones to work properly.
